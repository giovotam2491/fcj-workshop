During my internship at **Amazon Web Services Vietnam**, as part of the **Workforce Bootcamp – First Cloud AI Journey (FCAJ)** program, from **17/04/2026** to **10/07/2026**, I had the opportunity to learn, practice, and apply the knowledge acquired at university to a real-world working environment.

Over these 12 weeks, I progressed from AWS fundamentals (IAM, EC2, VPC, S3, RDS/DynamoDB) to designing, building, and deploying a complete serverless system — the **Internal Document Q&A Chatbot using Retrieval-Augmented Generation (RAG) on AWS Serverless**, powered by Amazon Bedrock Knowledge Base, Titan Text Embeddings V2, and Amazon Nova Lite. Through this project I improved my skills in cloud architecture design, serverless backend development (Lambda, API Gateway, DynamoDB), authentication with Amazon Cognito, Infrastructure as Code, monitoring, security review, and bilingual technical documentation.

In terms of work ethic, I always strived to complete tasks on schedule, complied with the program's working regulations, and actively engaged with my mentor and fellow interns to improve work efficiency.
