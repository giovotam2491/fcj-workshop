#### Tổng quan

Frontend là ứng dụng **React + Vite (TypeScript)** với giao diện 3D glassmorphism, chạy trên API và Cognito User Pool đã tạo ở phần trước. Trong phần này bạn sẽ chạy thử ở local để kiểm tra toàn bộ hệ thống, và có thể triển khai lên **S3 + CloudFront** để có mô hình hosting production.
