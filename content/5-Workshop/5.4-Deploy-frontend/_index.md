#### Overview

The frontend is a **React + Vite (TypeScript)** single-page application with a 3D glassmorphism UI, built on top of the API and Cognito User Pool provisioned in the previous section. In this section you will run it locally to validate the full stack, and optionally deploy it to **S3 + CloudFront** for a production-style hosting setup.
